  • After adding CogniPath, restart Chrome browser.
  • Open devtools panel.
    • Right-click anywhere on the webpage.
    • Click on Inspect.
  • On right side of Elements tab, click on CogniPath tab. It will be the last tab, if not visible then expand sidebar.
  • If xpath not generating or any issue,
    • Close the current tab and try different website.
    • Make sure there is a url in address bar as it might not support on blank new tab.
    • Chrome extensions won't support on chrome store.
